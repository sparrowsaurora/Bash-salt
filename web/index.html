Why I Made It

I made Bash-salt as a way to streamline my development process.
when i began using a terminal, i found i set up a lot of shortcuts and tools
slowly it got bigger and bigger and it got harder and harder to set-up on different computers.
So i decided to make a tool that would automatically set up my shortcuts and tools on any new pc with a single download.
Eventually I realised if it's a useful too for me, it might help other developers.
